## Key Findings

### 1. Zig @Vector Usage
- Use `@Vector(N, T)` where N is power of 2 (4, 8, 16)
- `@reduce(.Add, a * b)` for dot products
- `@splat(value)` for broadcast operations
- `@select(mask, a, b)` for conditional moves

### 2. Cross-Platform SIMD
- `std.simd.suggestVectorLengthForCpu()` for auto-detection
- ARM NEON:4 f32 elements per vector
- AVX2: 8 f32 elements per vector
- AVX-512: 16 f32 elements per vector

### 3. Memory Layout
- Cache-blocking with BLOCK_SIZE =32 (fits in L1)
- Align allocations to 32 or 64 bytes
- Row-major preferred for attention

### 4. Threading
- Use `std.ThreadPool` for parallel work
- Partition by rows/heads/layers
- Thread-local arenas for thread safety
